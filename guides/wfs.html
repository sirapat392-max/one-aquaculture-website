WFS — กุ้งถ่ายขาว
(White Feces Syndrome)

อุจจาระขาวหรือเหลืองลอยน้ำ กุ้งกินอาหารลด FCR สูง โตช้า — เกิดจากหลายปัจจัยร่วมกัน ส่วนใหญ่สัมพันธ์กับ EHP และ Vibrio

🦠 สาเหตุ: EHP + Vibrio + organic ⏱ DOC เสี่ยง: 30–60 วัน 📉 ผลกระทบ: FCR สูง โตช้า

🔑 สรุปสำคัญ

  • WFS ไม่ใช่โรคเดี่ยว — เป็นอาการจากหลายสาเหตุ: EHP, Vibrio, Gregarine, สาหร่ายสีเขียวแกมน้ำเงิน
  • อาการเด่น: อุจจาระขาว/เหลืองลอยน้ำ, กินอาหารลด, FCR สูง, โตช้า
  • ไม่ตายเร็ว แต่ถ้าปล่อยไว้ผลผลิตตกหนักและอาจเกิด secondary infection
  • แก้ที่ต้นตอ: ควบคุม EHP, ลด organic load, เพิ่มโปรไบโอติก

🔬 สาเหตุ (ซับซ้อนและหลายปัจจัย)

WFS เกิดจากความเสียหายของ ท่อย่อยอาหาร (midgut tubule) ทำให้เยื่อบุลำไส้หลุดลอกปนมากับอุจจาระ เห็นเป็นสีขาวหรือเหลือง

EHPปรสิตทำลายเซลล์ตับอ่อน ลดการย่อยอาหาร → midgut tubule เสียหาย
Vibrio spp.ฉวยโอกาสทำลายลำไส้ที่อ่อนแอจาก EHP อยู่แล้ว
Gregarineปรสิตในลำไส้ ดูดสารอาหาร ทำให้เยื่อบุลำไส้อักเสบ
Cyanobacteriaสาหร่ายสีเขียวแกมน้ำเงินผลิต toxin ทำลาย epithelium ลำไส้
Organic overloadน้ำเสีย อินทรีย์สูง กระตุ้นให้ Vibrio และ pathogen เพิ่มจำนวน

🩺 อาการที่พบ

  • อุจจาระขาว เหลือง หรือครีม ลอยอยู่บนผิวน้ำหรือขอบบ่อ
  • กุ้ง กินอาหารลดลง feeding tray มีอาหารเหลือ
  • FCR สูงขึ้น — กินมากแต่ได้น้ำหนักน้อย
  • โตช้า CV อาจสูงขึ้น
  • ตับอ่อนบางครั้งมีสีผิดปกติ (ขาว/เหลือง) คล้าย EHP
⚠️ WFS อาจเป็นสัญญาณว่า EHP ระบาดในบ่อแล้ว — ควรส่งตรวจ PCR ยืนยัน

🛡️ การป้องกัน

  • ป้องกัน EHP — ตากบ่อ โรยปูนขาว PCR ก่อนปล่อย (ดูคู่มือ EHP)
  • ใช้โปรไบโอติก สม่ำเสมอ ทั้งในน้ำและผสมอาหาร
  • รักษาคุณภาพน้ำ — DO สูง, ลด organic load
  • อย่าให้อาหารเกิน — ลด waste ที่เป็นอาหาร Vibrio และ Gregarine
  • ตรวจ Vibrio count ในน้ำสม่ำเสมอ

🔧 การจัดการเมื่อพบ WFS

  1. ลดอาหาร 30–50% ทันที — ลด organic load และลดโอกาส pathogen เพิ่มจำนวน
  2. เพิ่มโปรไบโอติก ในน้ำและผสมอาหาร — ทุกวันแทนทุก 3 วัน
  3. เปลี่ยนน้ำ 10–20% เพื่อลด organic load และ pathogen load
  4. ส่งตรวจ PCR ตับอ่อนและ midgut — ยืนยัน EHP และ/หรือ Vibrio
  5. ถ้า Vibrio สูง ปรึกษาสัตวแพทย์ — อาจพิจารณายาปฏิชีวนะตามคำสั่ง
  6. เพิ่ม vitamin C ผสมอาหาร — เสริมภูมิคุ้มกันกุ้ง
✅ ถ้ากุ้งถ่ายขาวแล้วกลับมากินอาหารปกติหลังลดอาหาร 2–3 วัน มักหมายความว่าเกิดจาก overfeeding ไม่ใช่ EHP

📚 คู่มือที่เกี่ยวข้อง

EHP ไมโครสปอริเดีย Vibriosis — แบคทีเรีย Vibrio โปรไบโอติกในบ่อกุ้ง คุณภาพน้ำบ่อกุ้ง

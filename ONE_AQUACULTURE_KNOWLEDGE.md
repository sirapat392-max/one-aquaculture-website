## 3. ฐานข้อมูลโรคกุ้ง (11 โรค)

### 3.1 โรค EMS / AHPND (โรคตายด่วน)
- **สาเหตุ:** Vibrio parahaemolyticus สายพันธุ์ที่มี pirAB toxin
- **ความรุนแรง:** สูงมาก
- **onset:** เฉียบพลัน 2–5 วันหลังสัมผัสเชื้อ เกิดใน DOC 7–35
- **อายุเสี่ยง:** กุ้งอ่อนมาก PLx–DOC 35
- **keyDifferential:** ตับตับอ่อนซีด/หดตัว + กระเพาะว่าง + DOC < 35

### 3.2 โรคไวรัสตัวแดงดวงขาว (WSSV)
- **สาเหตุ:** White Spot Syndrome Virus (dsDNA)
- **ความรุนแรง:** สูงมาก
- **onset:** เฉียบพลัน 3–10 วัน
- **keyDifferential:** จุดขาว 0.5–2 มม. บนเปลือก + ตัวแดง/ชมพู พบทุกอายุ

### 3.3 โรค EHP (ไมโครสปอริเดียน)
- **สาเหตุ:** Enterocytozoon hepatopenaei
- **ความรุนแรง:** ปานกลาง–สูง
- **onset:** ค่อยๆ สะสม อาการชัดหลัง DOC 30–60
- **keyDifferential:** โตช้า CV > 30% แต่ไม่ตายเฉียบพลัน

### 3.4 โรคอุจจาระขาว (WFS)
- **สาเหตุ:** Vibrio spp. + EHP (มักพบร่วมกัน)
- **ความรุนแรง:** ปานกลาง
- **keyDifferential:** เส้นอุจจาระขาว/เหลืองลอยบนผิวน้ำยาวๆ

### 3.5 โรคเหงือกดำ / เหงือกน้ำตาล
- **สาเหตุ:** คุณภาพน้ำเลว (Fe สูง, pH ผิด, แบคทีเรีย)
- **ความรุนแรง:** ปานกลาง
- **keyDifferential:** เหงือกดำ/น้ำตาลใต้กล้องจุลทรรศน์ ไม่มีจุดขาว

### 3.6 โรคเปลือกนิ่ม / ไม่ลอกคราบ
- **สาเหตุ:** แร่ธาตุขาด (Ca, Mg, P) หรือน้ำคุณภาพต่ำ
- **ความรุนแรง:** เบา–ปานกลาง
- **keyDifferential:** เปลือกบางนิ่ม ลอกคราบไม่สำเร็จ ไม่มีสัญญาณโรค

### 3.7 โรควิบริโอซิส / กุ้งเรืองแสง
- **สาเหตุ:** Vibrio harveyi
- **ความรุนแรง:** ปานกลาง–สูง
- **keyDifferential:** กุ้งเรืองแสงในที่มืดกลางคืน (ชัดเจนที่สุด)

### 3.8 โรคหัวเหลือง (YHD)
- **สาเหตุ:** Yellow Head Virus (YHV)
- **ความรุนแรง:** สูงมาก
- **keyDifferential:** หัวเหลืองชัด ต่างจาก EMS ที่ตับขาว

### 3.9 โรค IHHNV (กุ้งแคระผิดรูป)
- **สาเหตุ:** IHHNV (ssDNA parvovirus)
- **ความรุนแรง:** ปานกลาง
- **keyDifferential:** หนวดผิดรูป + ตัวเตี้ย (Runt-Deformity Syndrome)

### 3.10 ลำไส้อักเสบแบคทีเรีย / กุ้งฝ้าย
- **สาเหตุ:** Bacillus spp., แบคทีเรียกลุ่ม Gram+
- **ความรุนแรง:** เบา–ปานกลาง
- **keyDifferential:** ลำไส้ขาวขุ่นมองผ่านเปลือก

### 3.11 กล้ามเนื้อตาย / กุ้งไข้ขาว
- **สาเหตุ:** ความเครียด (อุณหภูมิ, ออกซิเจนต่ำ, จับกุ้ง)
- **ความรุนแรง:** ปานกลาง
- **keyDifferential:** กล้ามเนื้อหางขาวขุ่น (ภายในเนื้อ ไม่ใช่เปลือก)

## 8. ระบบข่าว

- **แหล่ง RSS 4 แหล่ง:** Hatchery International, GAA Advocate, Undercurrent News, Aquaculture North America
- **GitHub Actions:** `.github/workflows/update-news.yml` รันทุกจันทร์ 08:00 TH (UTC 01:00)
- **flow:** ดึง RSS → กรอง aquaculture keywords → เรียงตาม pubDate → เลือก 12 บทความ → แปลเป็น Thai → บันทึก `news-data.json`
- **Cache:** localStorage key `one_news_cache`, TTL 6 ชั่วโมง
- **Auto-refresh:** server.js รีเฟรชอัตโนมัติทุก 24 ชั่วโมง + 5 วินาทีหลัง startup

## 9. ระบบ AI วินิจฉัย

### Input
- อาการที่เลือก (checkboxes)
- รายละเอียดบ่อ (text)
- รูปภาพ (ไม่บังคับ, max 5MB) → ส่งเป็น base64

### กลยุทธ์วินิจฉัย
- DOC < 35 วัน → สงสัย EMS ก่อน
- DOC > 50 วัน → WSSV / WFS / EHP
- ตาย 3–10 วัน → WSSV หรือ YHD
- โตช้า CV สูง ไม่ตาย → EHP
- กุ้งเรืองแสง → Vibriosis
- หนวดผิดรูป ตัวเตี้ย → IHHNV

### Output (JSON format)
```json
{
  "topDiagnosis": { "nameTH": "", "nameEN": "", "confidence": 85, "reasoning": "" },
  "differentials": [{ "nameTH": "", "nameEN": "", "confidence": 40 }],
  "severity": "รุนแรงมาก/รุนแรง/ปานกลาง/เบา",
  "immediateAction": "",
  "treatment": "",
  "prevention": "",
  "relevantProducts": ["ชื่อผลิตภัณฑ์ ONE"],
  "needVet": false,
  "disclaimer": ""
}
```

### กฎ needVet
- `true` เฉพาะ severity = "รุนแรงมาก" หรือ "รุนแรง"
- `false` สำหรับ ปานกลาง / เบา
- ปัญหาน้ำ → ขึ้น needVet ตาม severity ปกติ (vet ช่วย confirm ได้)

### Rate Limit
- 10 ครั้ง / IP / วัน (server-side in-memory Map)
- localStorage key `dx_usage` ทำ client-side UI
